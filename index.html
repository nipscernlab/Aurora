TCMM Terminal
Welcome to the TCMM Terminal!
To create a new Processor project, click New Project. Give your project a name and choose an installation location. The Processor Hub button will be enabled and the .spf project will open automatically. If you already have a project, click Open Project.
Click Processor Hub to create a processor using your predefined parameters. Develop your processor software in C± inside the .cmm file located in your project’s Software folder. When you modify a file, remember to save it to persist changes.
Next, configure the build settings in Processor Config — select which processor to compile, set clock cycles and simulation frequency.
When ready, run a step-by-step build or use Full Build to compile everything and open the waveform in GTKWave. Watch the build logs live in the terminals: TCMM, TASM, TIVERI, TWAVE. Clear any terminal using Clear Terminal.
Need help? The AI Assistant can speed things up. Click AI Assistant and choose a model: ChatGPT, Claude, or DeepSeek. Inspect your project structure with Open Folder.
Want a quick project summary? Project Info or explore more options in Menu.
