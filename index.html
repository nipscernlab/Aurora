TCMM Terminal
Welcome to the TCMM Terminal! To create a Processor project, click on New Project. Give your project a name and select the installation location. Then, the .spf project will open automatically and the Processor Hub button will be enabled. If you already have a created project, click on Open Project
Click on Processor Hub to create a processor with all parameters pre-defined by you. Program your processor using the CMM language in the .cmm file within the Software folder of your project. When modifying a file, click on Save to save it.
Next, configure the compilation in the Processor Config button, setting up the processor to be compiled, the number of clocks and cycles (frequency) for simulation, as well as the compilation flags for cmmcomp, asmcomp, and iverilog.
After configuration, you can compile step by step or use the Compile All button to compile the entire project and view the simulation in gtkwave. Monitor the process log in the terminals TCMM, TASM, TIVERI, TWAVE & TPRISM. Clear each terminal individually with the Clear Terminal button.
To backup your currently open project, click on Backup. AI assistance may be helpful. Click the Toggle AI button to activate it. Select one of the models: ChatGPT or Claude. Check your project hierarchy in Open Folder.
If you want to switch the IDE theme between light and dark, click on Toggle Theme.
Want to view information about the current project? Click on Project Info. Information about the IDE version, Chromium, Node.js, Electron, and Windows is available in the Info Aurora button. There's more in Menu.
