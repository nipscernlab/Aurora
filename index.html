TCMM Terminal
Bem-vindo ao Terminal TCMM!
Para criar um projeto de Processador, clique em Novo Projeto. Dê um nome ao seu projeto e selecione o local de instalação. Em seguida, o projeto .spf será aberto automaticamente e o botão Processor Hub será habilitado. Se você já tem um projeto criado, clique em Abrir Projeto
Clique em Processor Hubpara criar um processador com todos os parâmetros pré-definidos por você. Programe seu processador usando a linguagem C± no arquivo .cmm dentro da pasta Software do seu projeto. Ao modificar um arquivo, clique empara salvá-lo
Em seguida, configure a compilação no botão Processor Config, configurando o processador a ser compilado, o número de clocks e ciclos (frequência) para simulação, bem como as flags de compilação para cmmcomp, asmcomp e iverilog
Após a configuração, você pode compilar passo a passo ou usar o botão Full Build para compilar todo o projeto e visualizar a simulação no gtkwave. Monitore o log do processo nos terminaisTCMM, TASM, TIVERI, TWAVE Limpe cada terminal individualmente com o botão Limpar Terminal A assistência de IA pode ser útil. Clique no botão Assistente de IA para ativá-lo. Selecione um dos modelos: ChatGPT, Claude ou DeepSeek. Verifique a hierarquia do seu projeto em Abrir Pasta
Deseja visualizar informações sobre o projeto atual? Clique em Informações do Projeto Há mais em Menu
